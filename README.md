# GEE Applications

Collection of scripts using Google Earth Engine (JavaScript/Python API) for geospatial analysis.

## Scripts
- [Temperature Anomaly](Temperature_Anomaly.py) — Temperature anomaly calculation and extreme detection
- [NDVI Phenoology Assessment ](NDVI_Phenology.ipynb) —  calculate the NDVI vegetation index with a cloud mask, and analyze its temporal variation (monthly, quarterly, and annually)

### Requirements
- Google Earth Engine account
- Python 3.11
- geemap
- Xee
- netCDF4
- xarray 
- rioxarray
